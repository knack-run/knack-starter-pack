# Worked examples

End-to-end examples demonstrating the pack's skills against realistic inputs.

- [`01-idea-clarifier-walkthrough/`](./01-idea-clarifier-walkthrough/): a rough indie-hacker brief about a pricing tool for early-career freelance graphic designers. Shows the round-trip questions (audience-narrowing rejection, demand for an audience-voice quote), the four-section output, and how the soft upgrade-path teaser surfaces only when the idea fits a Knack pack's domain.
- [`02-first-skill-template-walkthrough/`](./02-first-skill-template-walkthrough/): a request to scaffold a `weekly-review` skill for build-in-public Friday posts. Shows the kebab-case rejection (`weeklyReview` fails), the category-trigger rejection ("a skill for productivity" fails), the file scaffolded with `FILL IN:` markers preserved, and the next-actions list pointing at line ranges in the new file.
